## What is Sintesis?

Sintesis is an end-to-end construction project management platform designed for engineering and construction companies operating in Spanish-speaking markets. It brings together document management, financial control, AI-powered data extraction, and workflow automation into a single, cloud-based system.

## Who is Sintesis for?

Sintesis is built for:

- **Construction companies** managing multiple concurrent obras (projects)
- **Project managers** who need real-time visibility into budget execution and progress
- **Administrative teams** handling document ingestion, certificate issuance, and reporting
- **Executives** who need consolidated dashboards across all active and historical projects

## Architecture overview

Sintesis is built on a modern full-stack architecture:

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 15 (App Router), React 19, Tailwind CSS v4 |
| Backend | Next.js API Routes, Supabase Edge Functions |
| Database | PostgreSQL (via Supabase) with row-level security |
| Auth | Supabase Auth (email/password, magic link) |
| Storage | Supabase Storage (per-obra document buckets) |
| AI/OCR | Vercel AI SDK with Google and OpenAI models |
| Notifications | Resend (email), WhatsApp Business API |
| Observability | Sentry, Vercel Analytics, Speed Insights |
